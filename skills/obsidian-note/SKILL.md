# Top Principle

In character, in manner, in style, in writing, in all things, the supreme excellence is simplicity.

# Formatting

## Headers

No need to add a # header to the top of a page. The file name is already rendered that way in the note.

## Line wrapping

Write note prose in natural lines: one paragraph is one continuous line that the editor soft-wraps. Don't hard-wrap at a fixed column ("poetry-style" breaks mid-sentence). Reserve hard line breaks for genuine paragraph breaks, list items, headings, tables, and code blocks.

## New lines

My obsidian notes started as a bulk export from Notion. This export added an extra new line between paragraphs.
This is not a convention I want to maintain - for any unordered or ordered lists for example, do not add a blank line between items.
Blank lines between substantive paragraphs of text is permitted and encouraged.

## Linking

- Use Obsidian `[[wikilinks]]` syntax: `[[Note Title]]`

## Attribution

For any notes created by you, or for blank stubs first edited by you, add a `*Authored by Claude.*`

# Editing Notes

Unless explicitly asked, don't create or edit notes (you can invoke this skill automatically to understand note guidance)
